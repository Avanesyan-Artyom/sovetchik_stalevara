# Цифровой советчик сталевара

Система рекомендаций по дозировке ферросплавов для участка **печь-ковш** на сталеплавильном предприятии. Советчик предсказывает финальное химическое содержание нормируемых элементов (Cr, Mo, Ni, V, W) в стали и подбирает оптимальную комбинацию ферросплавов, при которой содержание каждого элемента попадает в интервал `(LowerLimit, Target)`.

---

## Производственный контекст

Цикл изготовления стали включает три ключевые стадии:

1. **Доменная печь** — плавка железной руды, получение чугуна  
2. **Дуговая электрическая печь (EOP)** — чугун + металлолом → полупродукт  
3. **Печь-ковш (LF/VD)** — полупродукт + ферросплавы → сталь заданного состава  

Качество стали определяется финальным процентным содержанием элементов. Сталевар вручную принимает решение о дозировке ферросплавов, что приводит к систематическому перерасходу. Советчик автоматизирует этот выбор.

---

## Данные

Источник — многолистовой Excel-файл с историей плавок участка печь-ковш:

| Таблица | Содержимое |
|---------|-----------|
| Table 1 | Метаданные плавок: номер, дата, вес стали, состав полупродукта (`Элемент_Last_EOP`) |
| Table 2 | Нормативные границы и фактические значения: `LowerLimit`, `Target`, `UpperLimit`, `Final` |
| Table 3 | Журнал событий (EventText) |
| Table 4–6 | Количество и наименование ферросплавов, добавленных в печь-ковш (`LFVD_*`) |
| Table 7 | Руководитель смены |
| Table 8 | Наименование и вес лома по стадиям загрузки |
| Table 9–10 | Ферросплавы и лом, добавленные в процессе ведения плавки (`PV__*`) |

Нормируемые элементы: **Cr, Mo, Ni, V, W**.

## Методология

### `eda.ipynb` — Анализ данных и экономический эффект

**Предобработка:**
- Объединение 10 листов Excel в единый датафрейм по ключу `HeatNo`
- Удаление дубликатов и признаков с нулевой дисперсией (неинформативные константы)
- Отбор признаков: состав полупродукта, вес плавки, вид и вес лома, дозировки ферросплавов

**EDA:**
- Матрица корреляций Пирсона; выделение высококоррелированных пар (порог 0.8)
- Тепловая карта по ключевым элементам и ферросплавам
- Гистограммы отклонений `Final − LowerLimit` (запас от нижней границы) и `Final − Target` (отклонение от цели)

**Оценка экономического эффекта:**
- Гипотеза: советчик поддерживает содержание элементов на уровне `(LowerLimit + Target) / 2`
- Расчёт фактического перерасхода каждого ферросплава относительно гипотетической цели
- Пересчёт перерасхода в деньги через коэффициент «кг ферросплава на 1% прироста в 100 кг металла» и рыночные цены ферросплавов

### `adviser.ipynb` — Модели и советчик

**Модели:**
- 5 независимых `LGBMRegressor` (по одному на каждый нормируемый элемент)
- Признаки X: состав полупродукта, вес стали, вид и количество лома и ферросплавов
- Целевые переменные y: `Cr_Final`, `Mo_Final`, `Ni_Final`, `V_Final`, `W_Final`
- Метрики: MAE, RMSE, R², ME (систематическое смещение)
- Разбивка train/test по временно́му принципу (хронологическая выборка)

**Советчик (`adviser`):**
- Принимает на вход известные на момент плавки данные (`Last_EOP`, вес, лом) и нормативные границы
- Перебирает все исторические уникальные комбинации ферросплавов из датасета
- Для каждой комбинации формирует вектор признаков, прогоняет через 5 моделей и проверяет условие: `LowerLimit ≤ predicted ≤ Target + tolerance` для всех элементов
- Возвращает первую (или лучшую по метрике близости к центру коридора) подходящую комбинацию

## Запуск

1. Поместите файл данных `PQM - podatki za pilotni projekt (OCR12VM) - ENG.xlsx` в корень репозитория  
2. Запустите `eda.ipynb` — получите предобработанный датафрейм и анализ  
3. Запустите `adviser.ipynb` — обучите модели и запустите советчик на тестовых плавках  

---

## Результаты

- Систематическое смещение (ME) всех пяти моделей близко к нулю (< 0.005%), что означает отсутствие значимого bias  
- Небольшое занижение предсказания безопасно для советчика: модель рекомендует чуть больше ферросплава, исключая брак по нижней границе  
- Расчётный потенциал экономии ферросплавов подтверждён по всем пяти элементам
